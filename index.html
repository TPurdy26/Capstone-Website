Basics of Web Code

Command Line Basics:

In order to code websites, first you have to know how to navigate the Command Line. This allows you to make directories, create files, and move through folders on your computer seamlessly.

Git Basics:

Git is a version control software that helps manage different versions of source code. Almost every development project uses Git in some way, so this is an important skill to familiarize yourself with.

HTML Basics:

The bare bones of any website is the HTML. It allows you to insert information and display it on a webpage. It uses tags to declare different elements.

CSS Basics:

The CSS is responsible for styling a website’s HTML by using an element’s class or tag. For the course taken, I mainly used flex to style webpages

Javascript Basics:

In order for a website to be interactive, you need javascript. Using javascript gives you the power to manipulate HTML and CSS, read for “event” like clicks and hovering, and allows the developer to add logic to their website.
