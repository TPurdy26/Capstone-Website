HTML Basics

  • Almost all html elements are content surrounded by opening and closing html tags
  • Opening tag: <> (aka <[keyword]>)
  • Closing tag: (aka )
  • Elements - like containers for content
  • All html files need a boilerplate to tell the computer what version of html they should use to run it
  • “[Shift] !” generates the boilerplate in VS Code automatically
  • "head" - this is where you put important metadata
  • The “title” is human readable and what is displayed in the browser tab
  • The “body” is where all of the content visible to the viewers will be
  • "p" - makes paragraphs
  • "h[insert number]" - headings
  • "strong" - makes it bold
  • "em" - makes things italicized
  • "!— [some text] —" - comments
  • "ul" - unordered list
  • "ol" - ordered list
  • "li" list items
  • An “alt” attribute is a piece of text that will be shown if an image does not load
