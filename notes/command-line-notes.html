Command Line Basics

  • “mkdir” - makes a new directory
  • “pwd” - shows your current directory
  • “cd” - can move you through a file path
  • “ls” - shows what’s in the current directory
  • “rm” - removes things (can remove directories with “rm -r”)
  • “.” - directory you’re already in
  • “..” - previous directory
  • “cp” - copies a file (ex: “% cp quotes.txt thesis/quotes.txt)
  • “-r” - recursive (by putting “cp -r” before a directory it would copy a directory and all of its contents)
  • “mv” - moves files (ex: mv [old] [new])
  • “*” - can work as any character ( ex: “*.txt” would retrieve all .txt files)
