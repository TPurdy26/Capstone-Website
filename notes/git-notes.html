GIT Basics

  • “git clone [enter what you copied from your repo on github]” - connects the repo to your local machine
  • “git add [file name]” - adds file in question to the staging area in git (basically a waiting area for the files until you choose to commit them)
  • “git status” - shows the current status of your files in git
  • "git commit -m [message about what you changed]" - commits file in question to git
  • “git log” - shows a history of the repos changes
  • “git add .” - adds all current directory items to the staging area
  • “git push” - pushes all changes to github
