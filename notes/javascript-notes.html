Javascript Basics

  • Every line should end with “;”
  • Put “let” before a variable to allow it to be reassigned (let coolNum = 3)
  • Put “const” to stop it from being reassigned mid-code (ex: const coolNum = 3)
  • You don’t need to put “let” before a variable if you’re changing one that already exists
  • prompt(“Enter what you want to say”) - asks for user input
  • “typeof” shows what type of thing a variable is
  • Number(variable) changes a string into a number
  • Variable ++ (adds one to the variable)
  • Variable - (subtracts one from the variable)
  • >, <, <=, >= (does what you’d think)
  • Bitwise Operators:
  • AND (&)
  • OR (|)
  • XOR (^)
  • NOT (~)
  • Use “node” in your terminal to run javascript in terminal
  • Can put variables inside of strings using ${variable} (can also insert other stuff like expressions
  • Can start a new line and it will show up on the output
  • “&&” is and
  • “||” is or
  • function syntax: function funcName(parameter) { [code]}
